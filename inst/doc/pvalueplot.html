pvalueplot example

James Black

2016-09-04

Below is example use of the function pvalueplot.

library("episheet")

Look at one estimate

episheet::pvalueplot(
  est1.ll = 0.7,
  est1.ul = 1.1,
  xlabel = "Odds ratio"
)

Look at two estimates

episheet::pvalueplot(
  est1.ll = 0.8,
  est1.ul = 3.8,
  est2.ll = 1.2,
  est2.ul = 2,
  label1 = "Estimate 1",
  label2 = "Estimate 2",
  xlabel = "Relative Risk"
)
## Confidence interval type is  95%CI
